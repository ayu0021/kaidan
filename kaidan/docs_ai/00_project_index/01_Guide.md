# AI Collaboration Guide

## Project Type

This is a Unity project.

## Current AI Scope

AI is allowed to help with:

- code review
- bug analysis
- logic review
- performance review
- compatibility review
- documentation generation
- small scoped code fixes after approval

## Out of Scope

Do NOT modify these unless explicitly approved:

- 3D assets
- art assets
- textures
- materials
- shaders
- scene layouts
- lighting
- animation clips
- animator controllers
- project-wide environment settings
- package versions
- input settings

## Required Workflow

1. Read the project index first
2. Read architecture notes
3. Create a plan before making changes
4. Provide multiple fix options
5. Wait for approval before editing code
6. Keep changes small and localized
7. Log all changes in docs_ai/04_fix_logs/

## Severity Levels

- Critical
- High
- Medium
- Low
- Suggestion

## Preferred Output

- Markdown report
- CSV summary
- Notion-compatible table export
